## Điểm nổi bật

- ⚡ **Tốc độ load cực nhanh** — static site không cần server
- 🌙 **Dark/Light mode** — chuyển đổi theo hệ thống hoặc thủ công  
- 📝 **Blog với Markdown** — viết và publish bài dễ dàng
- 🏷️ **Tags & Categories** — dễ dàng tìm kiếm nội dung theo chủ đề
- 🚀 **Auto-deploy** — push code là tự động lên GitHub Pages
