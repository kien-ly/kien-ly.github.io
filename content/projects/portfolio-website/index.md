## Tổng quan

Trang portfolio và blog cá nhân được xây dựng bằng **Hugo** — static site generator nhanh nhất hiện nay — kết hợp với theme **PaperMod** tối giản, hiện đại.

## Tính năng

- ⚡ Build siêu nhanh với Hugo
- 🌙 Dark / Light mode tự động theo hệ thống
- 📝 Viết blog bằng Markdown
- 🏷️ Tags & Categories để phân loại nội dung
- 🚀 Auto-deploy qua GitHub Actions

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Generator | Hugo v0.146+ |
| Theme | PaperMod |
| Hosting | GitHub Pages |
| CI/CD | GitHub Actions |

## Ghi chú

Deploy hoàn toàn miễn phí. Mỗi khi push code lên `main`, GitHub Actions sẽ tự build và publish.
