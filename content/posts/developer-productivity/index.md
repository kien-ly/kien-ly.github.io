## 1. Deep Work — Làm việc trong trạng thái tập trung tuyệt đối

Tôi block **3-4 tiếng sáng** để làm công việc quan trọng nhất. Không Slack, không email, không meetings.

Công cụ hỗ trợ:
- [Forest App](https://www.forestapp.cc/) — trồng cây ảo để tập trung
- `Do Not Disturb` mode trên macOS

## 2. Commit code mỗi ngày — dù nhỏ

Không quan trọng lượng code, quan trọng là **duy trì momentum**. Một ngày fix typo hay cập nhật docs cũng tính.

```bash
git add .
git commit -m "chore: cập nhật README"
git push
```

## 3. Đọc code nguồn mở

Mỗi tuần tôi dành 1-2 tiếng đọc source code của các project nổi tiếng. Học được rất nhiều pattern hay mà documentation không có.

## 4. Viết lại những gì đã học

Viết blog không chỉ để chia sẻ — nó giúp **củng cố kiến thức** của chính mình. Feynman Technique!

## 5. Review code của người khác chủ động

Đừng chờ được assign — chủ động review PR của teammates. Cách học nhanh nhất mà không ai dạy bạn.
